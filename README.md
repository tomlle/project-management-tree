# 案件管理資料のフォルダ構成

## 参考資料
作成資料のテンプレートなどを格納。

- 要件定義書サンプル

## 受領資料

受領資料を格納。サービス(SaaSなど)やクライアントからの資料を置く。

- [使用サービス名]
- [クライアント名]
- [パートナー名]

## 成果物

クライアントへの納品資料を格納。

ウォーターフローであれば以下のような構成になる。

- 基本設計
- 要件定義
- 運用設計

## 打ち合わせメモ

議事録などを格納

## 提案資料

主に企画フェーズで使用したプレゼン資料を格納。パワーポイントで資料作成することが多い。

## 検討資料

提案資料の作成で調査した内容や検討した内容をまとめた資料を格納。

## 管理資料

成果物の作成状況やアカウント管理資料を格納。
